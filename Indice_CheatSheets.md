# Índice de Cheat Sheets para Programadores

Colección de guías rápidas para herramientas esenciales de desarrollo.

- [[Terminal]] - Comandos de Bash/Zsh para navegación, gestión de archivos y sistema.
- [[Git]] - Control de versiones, ramas y flujo de trabajo.
- [[Docker]] - Gestión de contenedores, imágenes y Docker Compose.
- [[SSH]] - Conexiones remotas, gestión de claves y túneles.
- [[MacShortcuts]] - Atajos de teclado para productividad en macOS.
- [[Uv]] - Gestor de paquetes y proyectos Python ultrarrápido.
- [[Setup]] - Guía de instalación de herramientas esenciales (Homebrew).
- [[HTML]] - Estructura semántica, formularios, multimedia y accesibilidad.
- [[CSS]] - Flexbox, Grid, Container Queries y nuevas features.
- [[Javascript]] - Guía moderna de JavaScript (ES6+, Async, etc.).
- [[React]] - Hooks (useState, useEffect), Custom Hooks y ecosistema.
- [[Python]] - Tipado, Dataclasses, Async y características modernas (3.10+).
- [[n8n]] - Automatización de flujos de trabajo con Docker/OrbStack.
- [[Sql]] - Consultas, Joins, Normalización y Funciones de Ventana.

- [[Excel]] - Fórmulas avanzadas, tablas dinámicas y atajos Mac.
- [[GoogleSheets]] - Funciones cloud (QUERY, IMPORTRANGE) y colaboración.
- [[GoogleDocs]] - Redacción, formato inteligente y atajos.
- [[GoogleAppsScript]] - Automatización de notificaciones y scripts para Drive.
- [[CloudRun]] - Despliegue, gestión de tráfico y contenedores serverless en GCP.
- [[Firebase]] - Configuración de Firestore, CRUD y reglas de seguridad.
- [[CiCdCloudRun]] - Pipeline automático GitHub -> Cloud Run.

---
*Nota: Estos archivos se encuentran en la carpeta `CheatSheets`.*
